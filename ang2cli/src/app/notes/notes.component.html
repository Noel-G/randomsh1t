Notes

selector: simply defines the css selector for the component

A Component is simply a Directive which has a Controller and a View.
The View is the template and the Controller is the class.

ng-repeat is now *ngFor :
*ngFor="let x of myArray"

angular-cli commands:

  • ng new PROJECT_NAME
  • ng serve
  • ng g component nameOfComponent
  • ng g directive nameOfDirective
  • ng g service nameOfService
  • ng g pipe nameOfPipe
  • ng g class nameOfClass
  • ng g interface nameOfInterface

After creation of a directive :

  1. 1- import it in app.module.ts
  2. 2- declare it in @NgModule declarations (still in app.module.ts)
  3. 3- add it in HTML
